Features

Everything You Need to Ship Fast

Built by the community, for the community. Every feature is open, extensible, and hackable.

One-Click Deploy

Deploy your OpenClaw bot with a single command. Pre-configured for Vercel, Railway, and Docker.

npx create-openclaw@latest

Fully Customizable

Every aspect is configurable. Personality, responses, integrations — bend it to your will with simple YAML configs.

claw config --personality witty

Community Plugins

Extend with 50+ community plugins. From sentiment analysis to multi-language support — plug and play.

claw plugin add sentiment-ai

Privacy First

Self-hosted and transparent. Your data never leaves your server. Audit every line of code yourself.

claw audit --full-report

Developer-First API

Clean REST & WebSocket APIs. Type-safe SDKs for TypeScript, Python, and Go. Ship integrations fast.

import { Claw } from '@openclaw/sdk'

Multi-Platform

Deploy to Discord, Slack, Telegram, and the web. One bot, every platform. Write once, launch everywhere.

claw deploy --platforms all
How It Works

Three Steps to Liftoff

From zero to a deployed bot in under 5 minutes. Seriously.

01

Fork & Clone

Fork the OpenClaw repo, clone it locally, and install dependencies with a single command.

$ git clone https://github.com/openclaw/bot && cd bot && npm i
02

Configure Your Bot

Set your API keys, customize the personality, and pick your target platforms in claw.config.yml.

$ claw init --interactive
03

Deploy & Ship

Push to your repo and let CI/CD handle the rest. Your bot is live, globally distributed, and auto-scaling.

$ claw deploy --prod
